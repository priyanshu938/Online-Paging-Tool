The Online Paging Tool






About

The Online Paging Tool is a web app built using various web technologies. The app asks the use to enter the number of frames and the page reference string and when the user clicks on any of the four buttons : FIFO, LRU , LFU and OPTIMAL , the user sees the corresponding Paging Chart , total number of page faults and page hits.

How to use?

-> Enter the number of frames.
-> Enter the page reference string.
-> Click on any of the four buttons
   to get the desired output.
