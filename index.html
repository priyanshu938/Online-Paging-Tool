The Online Paging Tool






About

This is an Online Page Replacement tool developed by Priyanshu Tiwari, a Computer Science Engineering student of Amity University Lucknow Campus.The Online Paging Tool is a web app built using various web technologies. The app asks the use to enter the number of frames and the page reference string and when the user clicks on any of the four buttons : FIFO, LRU , LFU and OPTIMAL ( these buttons corresponds to the four paging algorithms ) , the user sees the corresponding Paging Chart , total number of page faults and page hits. Apart from that user can also review the tool by commenting their views in the comment section.

How to use?

-> Enter the number of frames.
-> Enter the page reference string.
-> Click on the any of the four types of buttons to get the desired output.
